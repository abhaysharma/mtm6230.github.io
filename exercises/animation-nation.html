        # Animation Nation Exercise

        # Introduction

        Web Animations are everywhere and are used for many different reason.

        # Objective

        For this exercise, find and identify 4 different uses for web animation on public web sites. Share your finding with a neighbor.

        Common uses of animation include:

        • to indicate change
        • to provide feedback
        • to attract attention
        • to entertain
        • to provide additional information

        # Bonus

        For a bonus exercise, learn how to create different animations by reviewing the following CodePen Tutorials. Make sure to Fork each CodePen tutorial before beginning.

        • Button Twitch
        • Pulsing Button
        • Button with Sliding Background
        • Infinite Wave
